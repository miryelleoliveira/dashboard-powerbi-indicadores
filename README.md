# 📊 Dashboard de Indicadores Operacionais


## 📌 Objetivo

Desenvolver um dashboard mais profissional para visualização e acompanhamento de indicadores internos, com foco em análise estratégica.


## 🛠️ Desenvolvimento

* Organização e tratamento da base de dados no Excel
* Estruturação das informações para análise
* Modelagem relacional no Power BI
* Criação de KPIs e medidas utilizando DAX
* Implementação de filtros e segmentações interativas


## 🚀 Tecnologias Utilizadas

`Power BI` • `Excel` • `DAX` • `Modelagem de Dados`


## 💡 Aprendizados

Evolução em tratamento de dados, modelagem e construção de indicadores voltados para tomada de decisão.
